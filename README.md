# Financial Transaction Analysis
Python project for financial data analysis and fraud detection

# Overview
This project analyzes financial transaction data using Python to identify spending patterns, detect potential fraud indicators, and generate business insights.

# Objectives
* Analyze customer spending behavior
* Detect high-risk transactions
* Perform customer segmentation
* Provide business recommendations

# Technologies
* Python
* pandas
* numpy
* matplotlib

# Features
* Data cleaning and preprocessing
* Spending analysis by category
* Fraud detection using statistical thresholds
* Customer segmentation

# Project Structure
financial-transaction-analysis/
-transaction_analysis.py
-transactions.csv
-requirements.txt
- README.md

# How to Run

1. Clone the repository:
   https://github.com/LesegoMasilela/financial-transaction-analysis.git

2. Navigate into the folder:
   cd financial-transaction-analysis

3. Install dependencies:
   pip install -r requirements.txt

4. Run the script:
   python transaction_analysis.py

# Key Insights
* High-value transactions may indicate fraud risk
* Spending patterns vary across categories
* Customer segmentation supports targeted decision-making

# Relevance to Fintech
This project demonstrates how Python can be used in financial systems to support fraud detection and customer analytics, which are critical in fintech environments.
